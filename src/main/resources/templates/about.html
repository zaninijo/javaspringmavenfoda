Sobre

Este é um projeto Spring Boot com Maven, SQLite e Thymeleaf

📚 Documentação

Este projeto foi criado com uma estrutura completa de Spring Boot, incluindo:

  • Controllers: Manipulam requisições HTTP
  • Services: Contêm a lógica de negócio
  • Repositories: Gerenciam acesso aos dados
  • Models: Representam as entidades do banco de dados
  • Templates: Views renderizadas com Thymeleaf

🔌 API REST Endpoints

POST /api/users - Criar novo usuário
GET /api/users - Listar todos os usuários
GET /api/users/{id} - Obter usuário específico
PUT /api/users/{id} - Atualizar usuário
DELETE /api/users/{id} - Deletar usuário

💾 Banco de Dados

O projeto utiliza SQLite como banco de dados. O arquivo nivel.db será criado automaticamente na raiz do projeto quando a aplicação iniciar.

🚀 Para Iniciar

Execute o comando:

mvn spring-boot:run

A aplicação estará disponível em: http://localhost:8080
